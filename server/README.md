# Frp 控制端

## 项目简介

这是一个基于 [gin-vue-admin](https://github.com/flipped-aurora/gin-vue-admin) 开发的 Frp 控制端系统，用于管理和监控 Frp 节点，提供完整的用户管理、节点管理、流量统计等功能。

## 主要特性

- 🚀 基于 gin-vue-admin 框架开发

## 技术栈

### 后端
- Golang
- Gin Framework
- Gorm
- MySQL

### 前端
- Vue3
- Element Plus
- Vite
- Pinia

## 快速开始

### 环境要求
- Go 1.18+
- Node.js 16+
- MySQL 5.7+
